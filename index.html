1. Informações gerais

Esta Política de Privacidade descreve como o desenvolvedor Igor Rodrigues Damiano (IRDBR) coleta, usa e protege as informações dos usuários ao utilizar os aplicativos disponíveis na Google Play Store.

  • Jogo Bee Happy

Ao utilizar qualquer um dos aplicativos listados, você concorda com esta política.

2. Dados coletados

Os aplicativos não coletam dados pessoais identificáveis diretamente. No entanto, podem utilizar serviços de terceiros como o Google AdMob.

  • Identificadores de publicidade (Advertising ID)
  • Dados de uso anônimos
  • Informações técnicas do dispositivo

3. Publicidade

Os aplicativos utilizam o Google AdMob para exibição de anúncios, que pode coletar dados para personalização.

4. Compartilhamento

Não compartilhamos dados pessoais. Apenas serviços terceiros como AdMob podem processar dados conforme suas políticas.

5. Segurança

Medidas são adotadas para proteger as informações, porém nenhum sistema é 100% seguro.
